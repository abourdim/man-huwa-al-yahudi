# « Qui est juif ? » — Traduction française

**Min Huwa al-Yahūdī ? · من هو اليهودي ؟**
*Qui est juif ?* — **Abdelwahhab al-Massiri** (1938–2008) — Dar al-Shorouk, Le Caire

Projet de traduction française intégrale et fidèle de cet ouvrage critique
d'al-Massiri sur la problématique de l'identité juive et la définition du Juif
qui structure le contrat social sioniste.

## Principes de traduction (hérités des projets jumeaux)

| Décision | Choix |
|---|---|
| Fidélité | **100 %**, aucun résumé |
| Translittération | Phonétique française (al-Massiri, *ḥulūl*, *ḥawsala*) |
| Notes du traducteur | Inline `[NdT : …]`, rares |
| Concepts clés | Italique avec glose |
| Typo | « », espaces fines avant `:` `;` `?` `!` |
| Usage | **Personnel uniquement** — al-Massiri d. 2008 |
